Theorie

Welche Positionen gibt es im Volleyball?

Positionen

Auf dem Volleyballfeld gibt es pro Mannschaft 6 Positionen. Diese sind von 1-6 gegen den Uhrzeigersinn durchnummeriert.


Rollen

Es gibt insgesamt 4 wichtige Rollen auf dem Feld:
- Einen Zuspieler
- Zwei Außenangreifer
- Zwei Mittelangreifer
- und einen Diagonalspieler
Für die Startaufstellung gibt es eine bestimmte Reihenfolge die einghalten werden muss. Diese lautet (Zuspiel -> Außen -> Mitte -> Diagonal -> Außen -> Mitte). Dabei ist egal von wo der Zuspieler startet, die Reihenfolge muss immer beachtet werden.

Beispiel: Zuspiel startet auf der 1.

Rotation

Wurde von der Mannschaft ein Punkt erziehlt, werden die Positionen im Uhrzeigersinn gewechselt. Dabei behählt jeder Spieler seine Rolle!

Beispiel für eine Rotation. Warum heißt das Läufersystem eigentlich Läufersystem? Das Läufersystem wird immer an dem Zuspieler ausgerichtet. Steht der Zuspieler auf der 1 wird das aktuelle System "Läufer 1" gennant.
